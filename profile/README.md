## üöÄ Flagship Open-Source Projects

### [Lilavati](https://github.com/vibz-art/lilavati)
**The Infrastructure of Sacred Time.** A high-performance, enterprise-grade Rust engine for Panchang computation. Validated against NASA JPL ephemeris and the Surya Siddhanta, Lilavati provides the calendar infrastructure layer for global applications requiring true astronomical exactitude.
* **Tech:** Rust, REST API, WebAssembly
* **Focus:** High-precision decimal arithmetic, memory safety, zero floating-point drift.

### [Sanskrit Foundation Models](https://github.com/vibz-art/sanskrit-models)
**Bridging Panini and Transformers.** Research and development into making Sanskrit a high-resource computable language. We are exploring techniques like custom tokenizers and task-specific lightweight architectures to encode the world's most rigorous formal grammar into modern LLMs.
* **Tech:** PyTorch, Custom Tokenization, MLOps
* **Focus:** Low-resource NLP, morphological parsing, efficient fine-tuning.
